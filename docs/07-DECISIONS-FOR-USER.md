## 판단이 필요한 것 (기본값으로 이미 동작함)

### 1. Apple 개발자 계정 (무료 vs 유료)
- 무료(현재 가정): 7일마다 재서명 필요, 푸시는 로컬 알림+백그라운드 갱신 (docs/06)
- 유료($99/년) 보유/구매 의사가 있으면 APNs 실시간 푸시로 업그레이드 가능 — 알려주시면 구현

### 2. 전력/헬스케어 신호의 한계 수용 여부
백테스트 결과(docs/04 §5) 전력/헬스케어는 타이밍 신호의 부가가치가 제한적이라
"거의 상시 보유 + 극단 국면에서만 현금 전환" 보험 모드로 설정했습니다.
이 두 섹터를 아예 "점수만 표시, 신호 없음" 모드로 바꾸는 선택지도 있습니다.

### 3. KIS 계정의 해외주식 시세 권한
미국 시세는 현재 yfinance로 수집합니다(무키, 안정적). KIS 해외시세를 쓰려면
해외주식 서비스 신청이 되어 있어야 합니다 — yfinance 유지로 충분하다고 판단하면 액션 불필요.
